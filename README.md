# ai-skincare-nutrition-chatbot
An AI-powered skincare and nutrition chatbot that combines NLP, machine learning, TF-IDF, Naive Bayes, fuzzy matching, and Streamlit to provide personalized wellness recommendations.
Project Overview

This project presents an Advanced NLP-Based Skincare and Nutrition Guidance System for Wellness, an intelligent chatbot designed to deliver personalized skincare and nutrition recommendations. The system combines Natural Language Processing (NLP), Machine Learning, and rule-based reasoning to understand user queries and provide accurate, context-aware responses.

The chatbot uses a Naive Bayes classifier trained on TF-IDF features for intent classification, while entity recognition and fuzzy string matching identify skincare ingredients, skin types, nutritional supplements, and other relevant entities from user input. A recommendation engine generates personalized skincare product suggestions by combining user-defined filters with cosine similarity over TF-IDF vectors.

The application is developed using Python and Streamlit, providing an interactive web interface where users can chat with the assistant in real time. The system also includes dashboards for data visualization, product and ingredient analytics, FAQ management, conversation logging, and fallback mechanisms for unsupported queries.

By integrating rule-based knowledge with machine learning techniques, the project demonstrates how AI can be applied to develop an intelligent wellness assistant that supports informed skincare and nutrition decisions while delivering an engaging user experience.
